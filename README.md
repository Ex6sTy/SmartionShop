# SmartionShop

`SmartionShop` — это учебный проект, демонстрирующий работу с объектно-ориентированным программированием (ООП) в Python. В проекте реализованы классы для управления категориями и продуктами, а также функциональность для работы с JSON-данными и тестирование с использованием `pytest`.

---

## 📋 Функционал

### **Классы**

1. **`Product`**:
   - Описывает продукт.
   - Атрибуты:
     - `name` — название продукта.
     - `description` — описание продукта.
     - `price` — цена продукта.
     - `quantity` — количество на складе.
   - Проверка корректности:
     - Цена и количество не могут быть отрицательными.
   - Строковое представление: `Product(name, price, quantity)`.

2. **`Category`**:
   - Описывает категорию продуктов.
   - Атрибуты:
     - `name` — название категории.
     - `description` — описание категории.
     - `products` — список продуктов.
   - Глобальные счётчики:
     - `category_count` — общее количество категорий.
     - `product_count` — общее количество продуктов.
   - Метод `load_from_json` для загрузки категорий и продуктов из JSON.

## 🛠️ Установка и запуск

### Установка
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/username/SmartionShop.git
   ```
2. Перейдите в директорию проекта:
   ```bash
   cd SmartionShop
   ```
3. Установите зависимости:
   ```bash
   pip install poetry
   poetry install
   ```

### Запуск
Для запуска основного скрипта:
```bash
python src/main.py
```

---

## 🧪 Тестирование

Для запуска всех тестов выполните:
```bash
pytest
```

Для проверки покрытия тестами:
```bash
pytest --cov=src --cov-report=term-missing
```
